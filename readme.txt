MagniFinance Invoice System for WooCommerce.

== Description ==

MagniFinance Invoice System enables you to create simplified and normal invoices using MagniFinance and WooCommerce. MagniFinance is an online invoicing software approved by the Portuguese Tax and Customs Authority, which you can have access to from anywhere.

Checkout our portfolio at: http://www.webds.pt

== Installation ==

1. Upload the entire `magnifinance-invoice-system` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. On your WooCommerce menu > MagniFinance
4. Configure your options
